<div>
  <p>
   The input file contains two integer numbers <strong>X </strong>and <strong>Y</strong>. <strong>X </strong>is the product code and <strong>Y </strong>is the quantity of this item according to the above table.</p>
</div>
<h2>Output</h2>
<div>
  <p>
